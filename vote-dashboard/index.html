VoteApp

Access the link below to vote RED, BLUE, or GREEN

http://meetup-vote-app-front.s3-website-us-east-1.amazonaws.com/

This is a simple demonstration of Lambda, a compute service from AWS that runs your code in response to events.

Lambda monitors a DynamoDB table, which is populated with votes generated by texting RED, BLUE, or GREEN to a apigateway endpoint by our friends When the table is updated, Lambda invokes code to query the table and aggregate the votes, which are used to create the graph above. Read more at https://churrops.io.
